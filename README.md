CDP Chatbot
This project is a chatbot designed to assist users with how-to questions related to Customer Data Platforms (CDPs) like Segment, mParticle, Lytics, and Zeotap. It provides step-by-step guidance on setting up and using these platforms effectively.

Features
Interactive Chat Interface: Users can type questions in a friendly UI and receive responses from the bot.

Preloaded CDP Knowledge Base: The chatbot is equipped with predefined answers to common queries related to Segment, mParticle, Lytics, and Zeotap.

Responsive Design: A visually appealing and user-friendly layout that adapts to various devices.

Project Structure
Files
index.html

Contains the structure of the chatbot's interface.

Links to the styles.css file for styling and script.js for interactivity.

styles.css

Provides the styling for the chatbot, ensuring a clean and professional look.

Features responsive design for better usability.

script.js

Handles the chatbot's functionality.

Includes predefined responses for various CDP-related queries.

Manages user input and bot responses.

How to Use
Open index.html in a web browser.

Type your question about Segment, mParticle, Lytics, or Zeotap in the input box.

Click the Send button or press Enter to submit your question.

The chatbot will respond with the relevant guidance or instructions.

Example Questions
"How do I set up a new source in Segment?"

"How can I create a user profile in mParticle?"

"What is the process for building an audience segment in Lytics?"

"How do I integrate data with Zeotap?"
